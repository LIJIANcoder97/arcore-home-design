# arcore-home-design
这是我的本科毕设，一个AR家具购物平台，详细视频介绍见img文件夹。

功能
- 基本电商功能
- AR家具展示，在实际房间环境中摆放家具模型
- 购物计划，一个将购物方案用文字图片和AR展示的功能，灵感来自宜家的体验式营销
